# SignLearn AI

## Overview

**SignLearn AI** is an AI-powered personalized tutoring platform designed to help **Deaf and Hard of Hearing (DHH) children** learn sign language through an engaging, adaptive, and inclusive experience.

The platform personalizes learning paths based on user progress and interaction, using gamification and AI-driven recommendations to make learning effective and enjoyable.
